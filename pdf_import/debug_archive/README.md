# PDF调试脚本归档

本目录包含PDF提取功能开发阶段的调试脚本，已完成使命但保留以供参考。

## 脚本说明

- `debug_pdf_text.py`: 输出PDF原始文本，用于分析提取规则
- `debug_missing_fields.py`: 检查特定字段的提取情况
- `debug_2-21.py`: 针对2-21号PDF的专项调试

## 使用方法

如需调试新的PDF格式，可参考这些脚本的实现方式。

```bash
python pdf_import/debug_archive/debug_pdf_text.py
```

## 归档原因

这些脚本是开发阶段的临时调试工具，当前PDF提取功能已稳定运行。保留这些脚本是为了：

1. 提供调试方法的参考实现
2. 在需要支持新PDF格式时可以快速复用
3. 保留项目开发历史

遵循YAGNI原则，将不再使用的代码移出主代码空间，但保留其参考价值。
